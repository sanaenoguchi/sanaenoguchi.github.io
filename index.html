Web Designer Sanae Noguchi

茨城県在住のWebデザイナー、野口早苗です。オンラインスクールでWeb制作について学び、アパレル業界から憧れていたWebデザイナーへと転職活動中です。これからさらにWeb制作の技術や知識を身につけて、Webサイト運用や問題解決のアドバイスもできるデザイナーになりたいと思っています。

Skills

Web

HTML/CSSでのコーディングができます。JavaScript、PHP、WordPressも現在勉強中です。

Design

ポップ、名刺、柄の図案、ちらしなどの紙媒体のデザイン制作をしておりました。アパレルの専門学校のスタイリスト科卒業し、ショップ店員やスタイリストなどの仕事も経験があります。

Sports

中学、高校時代はバドミントン部で県大会まで出場いたしました。山登り、スケートボードなどが趣味で、最近はマラソンが毎日の日課です。
